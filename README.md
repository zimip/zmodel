# zmodel

A simple, open-source model web-editor for use with my upcoming mod.

zmodel lets you create, edit, and preview models directly in your browser — no local setup required. It's designed to be lightweight, straightforward, and easy to contribute to.

---

## What it does

- Edit model files through a clean web interface
- Preview changes in real time
- Export models ready to use with the mod
- No installation needed — just open and start editing

---

## Why I made this

Building models for the mod shouldn't require complex toolchains or proprietary software. zmodel is my attempt at making that process accessible to anyone who wants to contribute, regardless of their technical background.

---

## Getting started
```bash
git clone https://github.com/yourname/zmodel.git
cd zmodel
# follow setup instructions here
```

Or just open `index.html` directly in your browser if no server is needed.

---

## Want to contribute a model?

That's exactly why this tool exists. If you make something you're proud of, I'd love to include it in the mod.

**To submit a model:**

1. Create your model using zmodel
2. Export the file
3. Open an [issue](https://github.com/yourname/zmodel/issues) or send a [pull request](https://github.com/yourname/zmodel/pulls) with your file attached
4. Add a short description of what it is and how you'd like it used

All contributions will be credited. Even rough drafts or experiments are welcome — don't be shy.
